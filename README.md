## Репозиторий

[https://github.com/vokerwave/solar-system](https://github.com/vokerwave/solar-system)

- Эллиптические орбиты

# Особенности реализации

## Орбитальная механика

Планеты движутся по эллиптическим орбитам с использованием эксцентриситета.

Формула малой полуоси:

b = a * √(1 - e²)

Где:

- `a` - большая полуось
    
- `b` - малая полуось
    
- `e` - эксцентриситет
    

---

## Процедурный космос

Генерируются:

- звезды
    
- мерцание
    
- туманности
    
- виньетка
    

Каждый запуск создает новый космос.

---

## Pixel-art rendering

Текстуры масштабируются без сглаживания:

```javascript
imageSmoothingEnabled = false;
```

## `planets.js`

Содержит массив `PLANET_DATA`.

Каждая планета имеет:

```javascript
{
  id,
  name,
  color,
  orbitRadius,
  size,
  orbitalPeriod,
  eccentricity,
  imgPath,
  imgPath2,
  rotInterval,
  info
}
```

# Ветки проекта

В репозитории используются разные ветки (`branches`) для отдельных этапов разработки и экспериментов.

Такой подход используется для:

- новых визуальных эффектов
    
- тестирования механик
    
- рефакторинга
    
- альтернативных версий рендера
    
- оптимизации анимации
    
- экспериментальных фич
    

Основная стабильная версия обычно находится в ветке:

```text
main
```

Дополнительные ветки позволяют разрабатывать новые возможности независимо от основной версии проекта.
